 3D Exporter
==================

Create your 3D scenes & animations with any 3D software and export them from 3D Max.


******************
     Version
******************

1.0


******************
   3DS MAX SUPPORTED VERSION
******************

3DS Max 2011 and above.

******************
    Installation
******************

STEP 1 - Look for A_Exporter.dle in the provided package. Select it and copy it.

STEP 2 - Install the plugin into 3Ds Max
You need to paste the file (A_Exporter.dle), inside the stdplugs folder into the installation directory of 3Ds Max.
If you have 3Ds Max started, reinitialise it. When you start it, you'll now be able to export in .EAD format from the Export menu. 

More info:
https://github.com/aumentia/Hello3DBrowser_iOS/wiki
